# Canteen Management System (React + Tailwind + Router)

## Run
1. Install dependencies:
   npm install
2. Start dev server:
   npm run dev

## Tech
- React (Vite)
- TailwindCSS
- React Router DOM
- React Icons

## Features
- Menu listing (map)
- Filters + search
- Cart state (useState + props)
- Ternary rendering
- Checkout -> Orders (in-memory demo)
